×

I created a sophisticated system to accurately determine the angle, center point, and head position of bolts on various surfaces using computer vision. Romero-ChatBot

×

I developed a sophisticated chatbot using a Retrieval-Augmented Generation (RAG) system, which integrates advanced web scraping, data indexing, and language models to deliver accurate, contextually relevant responses. The process involved gathering data through web scraping, splitting it into 10,000-word chunks with a 1,000-word overlap, embedding these chunks for machine understanding, and storing them in a Faiss vector store for efficient retrieval. Upon a user query, the system retrieves the most relevant data chunks and uses a Large Language Model (LLM) to generate precise answers. This project posed challenges such as ensuring data quality, optimizing chunk size, managing large datasets, and refining user experience. Through this work, I demonstrated my expertise in building scalable and efficient AI systems that provide a seamless user experience.
