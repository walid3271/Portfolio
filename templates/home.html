Portfolio

Face Detection Using URL

I engineered a custom model integrated into a FastAPI application that processes image URLs to deliver detailed recognition data. Users can submit one or more image URLs, and the API responds with the detected individual's name, prediction accuracy, and precise image coordinates. The project harnesses the capabilities of PyTorch and YOLOv8 to perform high-accuracy image detection and classification. This work exemplifies my ability to build and deploy advanced machine learning models, providing scalable and efficient solutions for image recognition tasks.

REPOSITORY

Floor Object Rooms and Bed Direction Identification

I developed an AI solution designed to assist users in identifying and analyzing floor plans based on the orientation and location of objects. This model is capable of detecting specific floor objects, such as where a sofa is placed, and determining the current and optimal locations for rooms like kitchens. The solution was built using PyTorch, Ultralytics, OpenCV, and pandas, demonstrating my ability to apply AI techniques to real-world spatial analysis challenges and provide actionable insights for floor plan optimization.

REPOSITORY

Face Recognition

I created a facial recognition system designed for secure login purposes, integrating advanced AI technologies. The system accepts an image URL and a user ID, then verifies the submitted image against registered facial data associated with the ID. The response is delivered in JSON format, indicating whether the user is valid. This project showcases my expertise in developing secure authentication systems using PyTorch, Dlib, and the FastAPI framework, emphasizing my skills in facial recognition and API development for real-time applications.

REPOSITORY

Find Direction Of A Bolt

I created a sophisticated system to accurately determine the angle, center point, and head position of bolts on various surfaces using computer vision. Leveraging two YOLOv8 models—one for object detection and another for oriented bounding boxes—I achieved precise identification and orientation of bolts. The system processes images via a FastAPI endpoint, returning the bolt's center and angle in a structured JSON format. Built on Ultralytics YOLOv8 and PyTorch, this project underscores my proficiency in deep learning, object detection, and deploying machine learning models in real-world applications.

REPOSITORY

Virtual Control

I developed a gesture-based control system that allows users to manipulate their computer's mouse and volume using only hand and finger gestures. This project utilizes the powerful capabilities of MediaPipe for real-time hand tracking and OpenCV for image processing. The system accurately detects and interprets specific gestures to control the mouse pointer and adjust the volume, providing an innovative and touchless way to interact with digital devices. This project highlights my skills in computer vision, machine learning, and the practical application of AI to enhance user interaction with technology.
